Вариант No5 Разработать абстрактный класс "Объект недвижимости" с методами: рассчитать налог, установить ставку налога. На его основе разработать классы: Земельный участок, Частный дом, Квартира, Дача. Программа должна позволять добавлять любой объект, задавать стоимость объекта, производить расчёт налога. Ставка задается фиксированная для каждого класса в процентах. Пользователь должен иметь возможность просмотреть все объекты или объекты для конкретного класса. Программа должна сохранять и загружать данные о каждом объекте в файл.
